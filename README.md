This project is an endless runner game inspired by Geometry Dash. This game will focus on a fast-paced movement,
precise timing.

The player's goal will be to survive as long as possible while avoiding and destroying obstacles and collecting points 
to increase their score. This game will be getting progressively harder

**Planned Features:**
* Physics-nased player movement
* Gravity switching mechanics that allows players to run on ceilings as well as ground
* A collectable orb system that grans players weapons to allow to destroy obstacles while moving
* system where collecting more points will make the game faster and perhaps more complex obstacle patterns
